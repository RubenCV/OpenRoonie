Project Vision

Our vision is to create Open Roonie, a light programming language that contains the basic elements of actual imperative procedural languages. It will use tools such as PLU (Python Lex-Yaccc) to implement lexic and syntax to the project. Also, it will feature Google Developer's Blockly and Python Anywhere Console.

Programming Language Objective

The project objective is to focus on developing a programming language that can be understoo by kids, teenages and adults that are not familiar to programming. The Open Roonie language will use blockly for the graphic interface, to achieve simplicity and ease of use.

Project Scope

The project's scope is to design a simple programming language in which the phases of lexic, syntax and semantic analysis are developed. On the other hand, it will make use of tools such as Blockly and Python Anywhere that allow the implementation of a friendly and easy-to-use interface.
